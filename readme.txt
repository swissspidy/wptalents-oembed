Add WP Talents to the list of available oEmbed providers

== Description ==

This plugin adds WP Talents to the list of available oEmbed providers.

Have a look at the [WP Talents API Help](http://wptalents.com/about/api/ "WP Talents API Help) for more information about the API.

== Installation ==

1. Upload `wptalents-oembed` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it
